# SMS GROUP TEST

MEAN fullstack example project 

## Getting Started

Need Node JS version >= 8 with NPM 


### Installing

SERVER setup
```javascript
npm install
npm start

```
CLIENT setup

```javascript
cd client
npm install
npm start
```

API will be running on port 3000 by default
and client will be at port 4200

api path for e.g : 
> http://localhost:3000/alllocation

frontend client ui : 
> http://localhost:4200/
